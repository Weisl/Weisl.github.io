Engine Guide

Simple Export is a more convenient way to use Blenders built in exporters with all current export formats (fbx, alembic, usd, obj, ...). The settings can be adjusted in the same way as you would configure Blenders built in presets and exporters.

My workflow

I shared my workflow and strategies in the Blender Conference 2024 Talk: Game Art Insights: Art, Tech and Teamwork, where I talk about using Blender in non Blender game pipelines. This addon will help you streamline some of the steps outlined in my presentation and automate it further.

Creating Custom Presets:

Simple Export uses BLenders build in Preset system. It detects pre existing presets and you can create them in the familiar ways: - Export Panel: Set poroperties and save the preset - Collection Exporter: Set poroperties and save the preset

Simple Export ships with 3 presets to export static meshes to Unity, Godot and Unreal Engine. Game Art Creation is not always a simple straight transfer of data. A Blender scene might contain high and low poly meshes to be backed additionally to the final game assets. It is therefore important that you can customize your export collections and make them work for all situations.

Use of Presets:

Other export addons often focus exclusevly on getting the final asset from Blender to the game engine. Simple Export intends to be more versatile and also cover other export needs like exports for baking or sending the data to another software.
