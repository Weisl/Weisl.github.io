I have always been interested in pipelines and coding and have been writing small scripts and addons to help with everyday tasks. Working on these tools in my spare time is also a welcome change from my everyday job. A lot of these tools are born from personal needs and try to solve real-world issues. I am polishing and improving some of these tools and making them available to you!

There is a huge difference between creating tools for yourself and making them accessible to others. Generalizing them to work in more generic cases, fixing all the bugs, creating a consistent UX/UI, and handling all the bugs are a lot of additional work.
