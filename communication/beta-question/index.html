Subject: Feedback Request: Simple Export Add-on Beta

Hi,

I’ve been working on the Beta for Simple Export, a Blender add-on designed to streamline collection exports for game and 3D pipelines, and I’d love your feedback!

What it does:

Simple Export provides an organized overview of all export collections and simplifies batch exporting in Blender. It builds on Blender’s existing collection exporter while keeping your files fully compatible—even without the add-on installed.

It supports all major export formats (FBX, USD, Alembic, OBJ, etc.) and makes it easy to maintain consistent export settings, batch export, customize presets, and manage export paths. For the full release, I plan to include presets tailored for Godot, Unity, and Unreal Engine.

This is version 0.6, and I’d love to hear your thoughts on it. Specifically:

  1. UX/UI: Is it clear how to use this add-on? How could I make it more intuitive and improve the user experience?
  2. Features: Are there any specific features you feel are missing?
  3. Marketing: Is the description clear and appealing? What could I change to make it more compelling for you?
  4. Ask for Help: If you like the add-on and see potential, would you be open to writing a short testimonial? It’s also perfectly fine to say, “I don’t want to” or “I’ll recommend it if it provides XYZ.”
