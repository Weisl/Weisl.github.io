Release Notes

1.2.2

This release is fixing some major hotkey issues.

Fixes

  • #310 Collider tools seems to destroy the default keymap for 'N' and 'T'
  • #308 Addon keymap changes are lost after restarting Blender
  • #315 Removing "Physics Material" and "Visibility Groups" hotkey assignments deletes the wrong hotkey.

Known Issues

  • #313 Hotkeys for "Physics Materials" and "Physics Groups" doesn't work when assigned to key "P" in Blender 3.2 (woks for 2.93, 3.3, 3.4)

1.2.1

This release focuses on bug fixes for Auto Convex/Convex on older Blender versions and cylindrical collider creation.

Fixes

  • #303 Support Convex and Auto Convex for older Blender versions (2.93)
  • #299 Cylindrical collider generation is broken in SELECTION mode GLOBAL space

1.2

This release focuses on updating the Auto Convex library and all settings associated with it.

Feuatures

  • #188 Add support for VHACD 4.1 Auto Convex
  • #29 Improve Algorithm for cylindrical collision creation.
  • #213 Expose defaults to the N-Panel: Use Modifier Stack, Transform Space, Creation Mode, Default User Group, Cylinder Orientation, Cylinder Segments, Sphere Segments
  • #284 Move display settings to preferences
  • #216 Add operator button to toggle wireframe display in collider object viewport display settings enhancement New feature or request
  • #217 Mesh to collider, should the parent name be used if a parent exists
  • #247 Add weld before inflate/deflate
  • #265 Code: Store addon specific settings in addon properties
  • #269 Add convex hull modifier after decimating convex collisions
  • #270 Import objs for V-HACD
  • #282 Cleanup Modal Inputs

Fixes

  • #288 Material list index not found
  • #262 Mesh to Collider: Should be smarter about prefix/suffix
  • #258 Rename when changing collider group
  • #237 Indicate Convert affects the entire Object and ignores Edit mode selection
  • #244 Convert to collider, doesn't set group object color

1.1

This is a major release, adding cool new features, simplifying the code base and improving the overall experience for the user a lot.

Feuatures

  • #61 Collision Creation: Create a Single Bounding Collision from the Selection
  • #152 Collision Creation: Add arbitrarily oriented minimum bounding box which allows you to create an automatically oriented box collider with the lowest volume possible.
  • #166 Restructure Preferences
  • #253 Presets: Default presets get automatically installed.
  • #166 Presets: Reorganize, renamed and cleaned up
  • #255 Collider Groups: Collider groups offer a more customizable way of grouping, selecting, organizing, and naming colliders according to their functionality. Their number is currently fixed at 3 groups.
  • #254 Auto Convex: The V-hacd exe gets shipped with the addon and is setup automatically. No more manual setup required.
  • #224 Center pivots of collider objects for all collider shapes.
  • #157 Physics materials: Rework of the physics materials system
  • #161 Physics materials: Create new physics material from the panels
  • #148 Add: Delete collider buttons to menu
  • #128 Change visibility to be a toggle instead of an action.
  • #122 Add x-ray toggle for collider creation.
  • #65 Rename Collider mesh data.
  • #251 Add regenerate name operation: The main goal of regenerating name is to update the collider names based on the selected preset quickly.
  • #170 Set colliders to not being rendered
  • #176 Add Physics material menu and Display Menu to customizable hotkeys.

Fixes

  • #177 Cleanup old preset naming.
  • #183 Fix error on unregister.
  • #165 Convert to Collider: Error when cancelling.
  • #141 Convert to Collider: Navigation doesn't work.
  • #242 Convert to Collider: Cancelling results in the loss of materials for the original objects.
  • #239 Convert to Collider: All Instances are converted not only selected.
  • #252 Convert to Mesh: No feedback for invalid input
  • #151 Use selection instead of active object if active object is not a valid target object.
  • #123 Modal collider creation doesn't stop, even if there are 0 colliders in the collider list.
  • #51 UX: Improve Delta Multiplication for number inputs (alpha, displace, decimate) and add Shift and Control for further control.
  • ...
