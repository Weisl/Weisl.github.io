Release Notes

Simple Export (Beta) v0.5

The initial Beta release is here, featuring all core functionality of the Minimal Lovable Product (MLP).

Features:

  1. Streamlined Export Management
    Centralized list with batch operations for paths, presets, and bulk exports, simplifying batch exporting in Blender.
  2. Comprehensive Format Support
    Handles all major export formats, including FBX, Alembic, USD, OBJ, glTF, STL, and PLY, without altering Blender's native functionality.
  3. Adaptable for Games and Movies
    Compatible with workflows for game engines like Unreal, Unity, and Godot, as well as movie pipelines.
