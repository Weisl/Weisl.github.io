Collider Groups

Collider groups can be used to group, select and name colliders according to their functionality or for easier organization.

Both, display name and prefix/suffix of the collider groups can be set Preferences > Presets > Collider Groups. They are stored and changed with the presets.

The settings of collider groups are part of the presets and can therefore be modified under the Presets tab in preferences. They are saved and loaded with all other preset settings. You can set and change the display name as well as the name used for the object naming in the preferences. These settings are part of the presets that can be modified in the preferences.
