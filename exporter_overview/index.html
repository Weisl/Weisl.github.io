Streamlined Collection Export for Blender

What is Simple Export?

Simple Export streamlines batch exporting in Blender by providing an organized overview of all export collections and simplifying the setup process. The add-on enhances Blender's built-in collection exporter but doesn’t modify its native functionality. This ensures .blend files remain fully compatible and exportable with the same settings—even without the add-on installed. Simple Export supports all major export formats, including FBX, Alembic, USD, OBJ, glTF, STL, and PLY.

This approach keeps your workflow efficient, team-friendly, and accessible across collaborators, while providing tools to simplify and speed up export configuration

Feature Overview

  • Blender-Compatible Exports: Uses Blender’s internal collection exporter, ensuring .blend files and export settings are fully shareable without the add-on.
  • Persistent Export Settings: Export settings are saved within Blender's Collection Exporter, ensuring they are stored with the file and can be repeated.
  • Batch Export: Export multiple collections or entire scenes in one go.
  • Collection Overview: View and manage all export collections in one place.
  • Configurable for Any Engine or Pipeline: Tailor export settings to meet the requirements of any engine or workflow, including Unreal Engine, Unity, Godot, or custom pipelines.
  • Support for Multiple Export Types: Works with all export formats supported by Blender, including USD, Alembic, glTF, FBX, OBJ, STL, and PLY.
  • Custom Export Paths: Easily define and adjust export paths for collections.
  • Consistent Output: Maintain uniform naming and folder structures for exports.
  • Customizable Presets: Apply reusable export settings across collections.
  • Flexible Naming Options: Adapt asset names to fit specific pipelines.
  • Auto Directory Management: Automatically create folders for exports if they don’t exist.

Why Use Simple Export?

Simple Export builds on Blender’s existing functionality to simplify and organize the export process. It’s designed to save time, reduce errors, and improve collaboration by maintaining compatibility with Blender's native tools.

  • Enhanced Workflow: Provides an overview of all export collections in a single panel.
  • Compatible: Ensures export settings are accessible in the .blend file, even for collaborators without the add-on.
  • Organized: Helps manage export paths, presets, and naming conventions with ease.
  • Efficient: Reduces manual setup for batch and individual exports.
